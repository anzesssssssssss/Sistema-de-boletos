# Sistema de entradas con QR

Esta app está lista para publicarse en GitHub Pages y usar Firebase como backend para sincronizar eventos y compradores.

## Publicar en GitHub Pages
1. Crea un repositorio en GitHub.
2. Sube estos archivos.
3. En GitHub, entra a Settings > Pages.
4. Activa GitHub Pages con la rama `main`.
5. Espera a que termine el deployment.

## Importante
- La página se abrirá desde `index.html`.
- La app necesita que en la sección de configuración de Firebase pegues el JSON del proyecto.
- La sincronización funciona en línea porque ahora usa Firebase Realtime Database.
